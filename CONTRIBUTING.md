# Contributing

기여해주셔서 감사합니다. 아래 가이드를 읽고 참여해주세요.

## 기여 방법

1. 이 레포를 Fork합니다.
2. 기능 브랜치를 만듭니다. (`git checkout -b feat/my-feature`)
3. 변경사항을 커밋합니다.
4. Fork한 레포에 Push합니다.
5. Pull Request를 생성합니다.

## 버그 리포트 / 기능 제안

[Issues](../../issues)를 통해 자유롭게 제보해주세요.

## 머지 정책

PR 리뷰와 머지는 관리자만 진행합니다.
기여는 누구나 환영하지만, 머지 여부는 관리자가 결정합니다.

## 커밋 컨벤션

```
feat: 새로운 기능
fix: 버그 수정
docs: 문서 수정
refactor: 리팩터링
chore: 빌드/설정 변경
```
